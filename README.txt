

This program (The-ONE-Simulator-For-ICNDTN) is the ONE simulator for information centric netwoking.

This program is modified at in the ONE simulator and based on IETF draft, "Extension of Probabilistic Routing Protocol using History of Encounters and Transitivity for Information Centric Network".

The ONE simulator URL: https://akeranen.github.io/the-one/
IETF draft, "Extension of Probabilistic Routing Protocol using History of Encounters and Transitivity for Information Centric Network" URL: https://tools.ietf.org/html/draft-chung-dtn-extension-prophet-icn-01
